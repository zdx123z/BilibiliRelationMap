## üìñ Project Overview

Welcome to BilibiliRelationMap! This project is a userscript designed to enhance your experience on the Bilibili website. Built using React, TypeScript, and Ant Design, this tool helps you manage and visualize your social connections on Bilibili in a modern and user-friendly interface.

### üåü Core Features

- **View Follower List**: Easily browse through your followers and check their details.
- **Manage Following List**: Access your following users and view mutual connections.
- **Interactive Relationship Map**: Visualize your relationships in a clear, interactive format.
- **Professional UI**: Enjoy a sleek design powered by Ant Design.
- **Responsive Design**: The application works smoothly on various screen sizes.
- **Smart Caching**: Uses LocalStorage to minimize API calls for better performance.
- **Performance Optimizations**: Experience fast loading with batch processing and pagination.
